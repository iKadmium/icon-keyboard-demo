Icon Keyboard MIDI Demo

This demonstrates the feedback bug in Icon keyboards.

Requesting MIDI access...

  1. First, please select the keyboard from the list above. If an iCON keyboard is connected, it will have been auto-selected for you.
  2. Next, turn the encoder knob on the keyboard to zero.
  3. Then, click the test button. That will send a CC message to set the knob to 100%.
  4. Observe that the LED encoders indicate that the value is now at 100% (value: 127).
  5. Next, turn the knob in either direction. Note that it has now been reset to 0.
